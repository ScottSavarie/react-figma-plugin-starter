# react-figma-plugin-starter

Boilerplate setup for developing React based Figma plugins. Comes with a basic example of publishing / subscribing to messages between App.tsx and code.ts.

## Requirements

- Node 14+

## Building the Plugin:

```
    $ npm install
    $ npx webpack
```
